Skenario Pengujian:

ViewModel:
- Register:
		- Ketika berhasil melakukan register
			- Memastikan api register terpanggil
         	- Memastikan mengembalikan DataResult.Success.
		- Ketika gagal melakikan register
			- Memastikan api login terpanggil
			- Memastikan mengembalikan DataResult.Error

- Login:
		- Ketika berhasil melakukan login
       		- Memastikan api register terpanggil
			- Memastikan response tidak null.
			- Memastikan mengembalikan DataResult.Success.

		- Ketika gagal melakikan login
			- Memastikan api login terpanggil
			- Memastikan mengembalikan DataResult.Error

- Story
	- Ketika berhasil memuat data story.
			- Memastikan api story terpanggil
			- Memastikan mengembalikan DataResult.isSuccess.
			- Memastikan data tidak null.
			- Memastikan jumlah data sesuai dengan yang diharapkan.

	- Ketika gagal memuat data story.
			- Memastikan api story terpanggil
			- Memastikan mengembalikan DataResult.Error.
- Upload
	- Ketika berhasil mengupload data story.
			- Memastikan api add-story terpanggil
			- Memastikan response tidak null.
			- Memastikan mengembalikan DataResult.Success.

	- Ketika gagal memuat data story.
			- Memastikan api add-story terpanggil
			- Memastikan response tidak null.
			- Memastikan mengembalikan DataResult.Error.

- Maps
	- Ketika berhasil memuat data story di map.
			- Memastikan api story terpanggil
			- Memastikan mengembalikan DataResult.isSuccess.
			- Memastikan data tidak null.

	- Ketika gagal memuat data story di map.

- Utils
    - Memastikan format tanggal sesuai dengan yang diharapkan
    - Memastikan jika input tanggal tidak sesuai format, maka akan dikembalikan tanggal tanpa pemformatan
    - Memastikan jika zona waktu tidak sesuai terdaftar, maka akan dikembalikan tanggal tanpa pemformatan
- Database
    - Ketika berhasil menyimpan data
        - Memastikan data saat ini sesuai dengan yg diharapkan
    - Ketika berhasil menghapus data
        - Memastikan data menjadi kosong
- Story
    - Ketika gagal memuat data story
        - Memastikan informasi data tidak dapat memuat data ditampilkan
    - Ketika data story kosong
        - Memastikan informasi data tidak dapat memuat data ditampilkan

        - Memastikan komponen recyclerview tampil
        - Memastikan data yang diharapkan tertampil
        - Memastikan data selanjutnya yang diharapkan juga tertampil

        - Memastikan tombol login disabled

         - Memastikan password diinput dengan benar

         - Memastikan tidak berpindah halaman
    - Ketika berhasil login
        - Memastikan email diinput dengan benar
        - Memastikan password diinput dengan benar
        - Memastikan tombol login enabled
        - Memastikan diarahkan ke halaman home
